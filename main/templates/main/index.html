payment Django+Stripe

REDME

Description

- The project was created as a test task.

- The task is to implement a payment system for goods using Stripe

Go to the admin panel:

- `http://127.0.0.1:8000/admin/`

Usage API

- `GET http://127.0.0.1:8000/api/`

HTML response

Request with the help of which you can get a simple HTML page with information about the selected Item and a Buy button. By clicking on the Buy button, a request for api/buy/{id} should occur, session_id should be obtained, and then a redirect to the Checkout form stripe should occur using the Stripe JS library.redirectToCheckout(SessionID=session_id)

- GET http://127.0.0.1:8000/item/{id}

Request with the help of which you can get a simple HTML page with information about the selected Order and a Buy button. By clicking on the Buy button, a request for api/buyorder/{id} should occur, session_id should be obtained, and then a redirect to the Checkout form stripe should occur using the Stripe JS library.redirectToCheckout(SessionID=session_id)

- GET http://127.0.0.1:8000/order/{id}`
